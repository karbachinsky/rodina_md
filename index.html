    description

    functions

    advantages

    To order this package is necessary to leave a minimum of information in a special form. To communicate with the manager the user can only leave one telephone number and order a call back. to help make the service better, the user can answer a few questions in the questionnaire. It is not necessary to write your proposal. Suffice it to point out that was not fully comfortable and informative.Access to these functions, contact information, settings, instructions and other pages is possible from any part of the site through the menu.

    Designed for Interactive Services
